# Mortar 语言指南

欢迎来到 Mortar 官方文档！Mortar 是一种专为游戏对话和文本事件系统设计的领域特定语言 (DSL)。

## 什么是 Mortar？

Mortar 是一种现代对话脚本语言，强调**文本内容与事件逻辑的严格分离**。与其他对话系统不同，Mortar 确保您的叙述内容保持干净，不被技术标记污染，同时仍提供强大的事件驱动功能。

## 主要特性

- **内容分离**: 叙述文本和系统事件的清晰分离
- **清晰语义**: 类似 Rust 的直观可读语法
- **程序友好**: 编译为 JSON，便于集成
- **静态类型**: 编译时类型检查，确保可靠性
- **事件驱动**: 基于字符索引的精确事件时序

## 谁应该使用 Mortar？

Mortar 非常适合：

- 🎮 **游戏开发者** 为 RPG、视觉小说或冒险游戏构建对话系统
- 📚 **互动小说作者** 创建分支叙述
- 🎬 **多媒体创作者** 将文本与音频、动画或视觉效果同步
- 🤖 **聊天机器人开发者** 设计结构化对话流程

## 设计哲学

> "Mortar 旨在实现文本内容与事件逻辑的严格分离。"

这一核心原则驱动着 Mortar 的每一个设计决策。您的写作者可以专注于创作引人入胜的叙述，而您的开发者可以实现丰富的交互功能 - 所有这些都不会相互干扰。
